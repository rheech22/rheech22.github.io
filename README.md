## 주안점

- 심플하고 절제된 반응형 디자인
- 위키 형식
  - 최근 업데이트된 문서 목록을 제공하는 메인 페이지
  - 문서에서 상위 문서를 확인하고 네비게이션할 수 있음
  - 문서에서 갱신 내역(commit, blame, date)를 확인할 수 있음
  - 인덱스 페이지 제공
  - Vimwiki와 연동
